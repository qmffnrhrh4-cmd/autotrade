# 🚀 AutoTrade - 시작 가이드

## ⚡ 원클릭 설치 및 실행

### 단 하나의 파일로 모든 것을 해결하세요!

```cmd
autotrade_setup.bat
```

**끝!** 🎉

---

## 📋 무엇을 할 수 있나요?

`autotrade_setup.bat` 하나로 모든 것을 할 수 있습니다:

1. **🔧 전체 설치** - 32비트 환경 생성 + 모든 패키지 설치
2. **📦 패키지 설치** - 이미 환경이 있다면 패키지만 설치
3. **🧪 로그인 테스트** - 키움 OpenAPI 로그인 테스트
4. **🚀 메인 프로그램** - AutoTrade 실행
5. **ℹ️ 환경 정보** - 설치 상태 확인
6. **🗑️ 환경 제거** - 완전히 삭제하고 재설치

## 🔧 문제 해결

### ❌ "No Qt bindings could be found"

**해결:** `autotrade_setup.bat` 실행 → 메뉴 [1] 선택

스크립트가 자동으로 PyQt5를 설치합니다.

### ❌ "No module named 'pydantic'"

**해결:** `autotrade_setup.bat` 실행 → 메뉴 [2] 선택

모든 필요한 패키지를 자동 설치합니다.

### ❌ 64비트 환경에서 실행 중

**확인:** `autotrade_setup.bat` 실행 → 메뉴 [5] 선택

현재 환경 정보를 확인할 수 있습니다.

**해결:**
- 메뉴 [6]으로 환경 제거
- 메뉴 [1]로 재설치

### ⚠️ 32비트 환경 생성 실패

**원인:** Anaconda가 64비트 전용일 수 있음

**해결책 1 (권장):**
```cmd
# 32비트 Python 직접 설치
# 1. https://www.python.org/downloads/ 접속
# 2. "Windows installer (32-bit)" 다운로드
# 3. 설치 후:

cd C:\Users\USER\Desktop\autotrade
C:\Python311-32\python.exe -m venv venv_32bit
venv_32bit\Scripts\activate
pip install -r requirements.txt
python test_login.py
```

**해결책 2:**
- 32비트 Anaconda 재설치
- 또는 koapy 서버 모드 사용 (자세한 내용: `docs/SETUP_32BIT_ENVIRONMENT.md`)

- **키움증권 OpenAPI**: https://www.kiwoom.com/

### 자주 묻는 질문

**Q: 왜 32비트가 필요한가요?**
A: 키움증권 OpenAPI는 32비트 ActiveX 컴포넌트입니다. REST API는 64비트 가능합니다.

**Q: 64비트 환경에서는 못 쓰나요?**
A: OpenAPI는 32비트 필수입니다. koapy는 서버 모드로 64비트에서도 사용 가능하지만, 32비트 Python이 설치되어 있어야 합니다.

**Q: 설치 시간이 얼마나 걸리나요?**
A: 인터넷 속도에 따라 5-15분 정도 소요됩니다.

**Q: 기존 Python 환경은 어떻게 되나요?**
A: 영향 없습니다. 새로운 `autotrade_32` 환경을 별도로 생성합니다.
